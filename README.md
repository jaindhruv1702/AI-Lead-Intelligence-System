# AI Lead Intelligence & Lead Scoring System

## Overview

This project automates lead qualification and prioritization using AI agents built with n8n. The workflow performs company research, lead enrichment, lead scoring, categorization, and automated stakeholder notifications.

## Problem Statement

Sales teams often spend significant time researching and qualifying incoming leads manually. This project reduces manual effort by automating lead analysis and prioritization using AI.

## Workflow Architecture

1. New Lead Submission
2. Company Research Agent
3. Lead Scoring Agent
4. Lead Categorization
5. CRM / Google Sheets Update
6. Team Notification

## Technologies Used

- n8n
- Google Gemini
- Google Sheets
- Gmail
- AI Agents
- Workflow Automation

## Features

- Automated company research
- AI-generated lead insights
- Lead qualification and scoring
- Hot/Warm/Cold lead classification
- Automated notifications
